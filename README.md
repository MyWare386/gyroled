What is gyroled
---------------------
Turn on a RGB led and a buzzer with a gyroscope or a RFID card /badge on arduino

<br>

Contents
-------------------

<br>

How to use
--------------------

**You need :**

- An arduino card
- A GY-521 module
- A RGB Led
- A RC522 RFID module
- A buzzer
